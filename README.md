# Simulação de Transações Bancárias em Kotlin

Este projeto consiste em um **script desenvolvido em Kotlin** que simula operações bancárias básicas, como **depósito** e **saque**.

O objetivo do projeto é demonstrar, de forma prática, conceitos de **lógica de programação**, **controle de fluxo** e **manipulação de valores**, simulando o funcionamento simples de uma conta bancária.

## Funcionalidades

- Realizar depósitos em uma conta
- Realizar saques
- Atualizar e exibir o saldo após cada transação
- Simulação simples de operações financeiras

## Tecnologias Utilizadas

- Kotlin

## Objetivo do Projeto

Este projeto foi desenvolvido com fins **educacionais**, para praticar conceitos de programação utilizando a linguagem Kotlin.
